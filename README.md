BrightTV Subscription & Usage Analysis
Project Overview

This project focuses on analyzing BrightTV’s user profiles and viewing behavior to support the CEO’s objective of growing the subscription base for the current financial year.

The analysis provides actionable insights to assist the Customer Value Management (CVM) team in improving user engagement, increasing consumption, and driving user acquisition.

Objectives
Analyze user and viewing trends
Identify factors influencing content consumption
Recommend content strategies for low-activity periods
Propose initiatives to grow the subscriber base

Data Source
Dataset provided via: portal.brightlearn.co.za
Note: All timestamps were originally in UTC and converted to South African time (SAST) for accurate analysis.

Tools & Technologies
Excel → Data cleaning and initial exploration
Databricks (SQL) → Data transformation and querying
Miro → Planning and structuring the analysis
Canva → Presentation design and storytelling
GitHub → Project documentation and version control
